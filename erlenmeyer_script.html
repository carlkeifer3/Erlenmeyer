Erlenmeyer allows you to create fully-functional Flask servers, complete with SQLAlchemy models, from a Core Data file.
It's Flask, SQLAlchemy, and Core Data. Get it? It's a chemistry pun. Get it?
docs
quickstart :: erlenmeyer script :: erlenmeyer module 
Usage: erlenmeyer [options]
            
Options:
  -h, --help            show this help message and exit
  -p PROJECT, --project=PROJECT
                        Location of the project to manipulate.
  -c COREDATAFILE, --coredata=COREDATAFILE
                        Location of the Core Data xcdatamodeld file.
  --sql=SQLLOGIN        The username:password combination for SQLALchemy.
  --primaryKey=PRIMARYKEY
                        The primary key of all Models. Default to UUID.
  --projectOwner=PROJECTOWNER
                        The owner to whom this project is copyrighted.
  --fileAuthor=FILEAUTHOR
                        The creator of this file. Defaults to current user.
  --force               If the project already exists, delete it.
