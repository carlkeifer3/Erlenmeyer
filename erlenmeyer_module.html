categories erlenmeyer.categories

The categories module provides a class and a series of functions for adding functions to modules and methods to classes at runtime.

  • categorize(destination, function, override = True)

    Adds the given function to the destiation, optionally overriding existing methods.

  • addCategories(destination, source, override = True, list = None)

    Adds the functions listed in list to the destiation, optionally overriding existing methods. If list is None, adds every method in the source.

CoreData erlenmeyer.CoreData

A dictionary object parsed from an .xcodedatamodeld file.

example: 
{
    "className": "MBMegaBitTrainer",
    "parentClassName": "database.Mode",
    "primaryKey": "uuid",
    "primaryKeyType": "String(256)",
    "attributes":
    [
        {
            "name": "uuid",
            "type": "String(256)"
        }
    ],
    "relationships":
    [
        {
            "name": "megabits",
            "className": "MBMegaBit",
            "inverseName": "trainer",
            "inverseClassName": "MBMegaBitTrainer",
            "isToMany": True,
            "inverseIsToMany": False,
            "inverseHasBeenHandled": False
        }
    ]
}

ext erlenmeyer.ext

The ext module provides methods to extend certain modules.

jinja2 erlenmeyer.ext.jinja2

The jinja2 module adds filters for use by the jinja2 templating engine.

  • camelcase(value)

    Transforms, for example, "megaBits" into "MegaBits".

  • underscore(value)

    Transforms, for example, "MegaBits" into "mega_bits".

Model erlenmeyer.ext.Model

The Model module adds methods to the SQLAlchemy Model class.

  • all(self)

    Returns a list of all models if this type. A class method.

  • get(self, identifier)

    Returns the model of this type with the given identifier. A class method.

  • __iter__(self)

    Allows dict(myModel) to return a dictionary representation of this object.

  • update(self, properties)

    Applies the given key-value pairs to this object.

  • save(self)

    Saves this object to the database and commits.

  • delete(self)

    Deletes this object from the database and commits.

SQLAlchemy erlenmeyer.ext.SQLAlchemy

The SQLAlchemy module adds methods to the flask.ext.sqlalchemy.SQLAlchemy class.

  • tableRelationship(self, sourceClassName, destinationClassName, primaryKeyName, primaryKeyType, inverseName = None)

    Used to map many-to-many relationships. Creates a reference table with two columns, one for each object's primary key. Then creates a relationship based on the table, and a backreference based on the given inverse name, if any.
